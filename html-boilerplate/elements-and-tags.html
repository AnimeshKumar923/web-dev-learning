Elements and Tags

Tags

HTML tags are those objects/things that have certain function in the language. There are two types, i.e., opening tag & closing tag

Opening tags

these are the opening tag which tells that a certain thing/work starts from here.

Closing tags

This closing tag which denotes that the the certain property being used, ends here.

Elements

Elements can be though like they are the thing which contains content inside them. For example, a box which contains things inside it.
